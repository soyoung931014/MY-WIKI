### 고차함수 map 특징
- map은 원본 배열을 탐색하면서 새로운 배열을 생성한다. 새로운 배열도 원본 배열의 길이와 같다. (참고로 필터는 아님)
```jsx
let a = [10,11,12,13,14,15]
let answer = a.map(el => {
  if(el%2===0) return el
})
console.log(answer)


// [10,undefined,12,undefined,14,undefined] ✅ 새로운 배열도 원본 배열의 길이와 같다.
```
